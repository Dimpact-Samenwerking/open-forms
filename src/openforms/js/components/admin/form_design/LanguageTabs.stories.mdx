# LanguageTabs

The language tabs can be used to differentiate between values for literals in different languages.
They are used in the Form Designer to enter translated values, e.g.: Form names, explanation
templates and submission templates.

export const Template = args => {
  return (
    <IntlProvider messages={{}} locale={'nl'} defaultLocale="nl">
      <FormContext.Provider
        value={{
          languages: languageMapping,
        }}
      >
        <LanguageTabs haveErrors={args.haveErrors}>
          {(langCode, defaultLang) => <div>{langCode}</div>}
        </LanguageTabs>
      </FormContext.Provider>
    </IntlProvider>
  );
};
