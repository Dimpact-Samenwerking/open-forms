==============
Javascript SDK
==============

The Javascript SDK is built to embed forms in various types of pages. It is used
internally in the Open Forms project for the public UI portion.

This documentation is intended for developers looking to integrate the SDK in their
project and maintainers of the SDK.

Storybook documentation
=======================

The SDK has dedicated (work-in-progress) documentation of the components it offers,
see `Github pages <https://open-formulieren.github.io/open-forms-sdk/>`_.
